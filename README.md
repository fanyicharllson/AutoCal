# AutoCal

AutoCal is a calendar generator web application that allows users to effortlessly create calendars without the hassle of manual design. Users can also download the generated calendar as a PDF.

## Features

- Easy-to-use interface for generating calendars
- Options to customize calendar layout and design
- Download the generated calendar as a PDF
- Supports multiple languages
- Responsive design for use on various devices

## Technologies Used

- **TypeScript** (77%)
- **JavaScript** (14.9%)
- **HTML** (5.3%)
- **CSS** (2.8%)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have the following installed on your machine:

- Node.js
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/fanyicharllson/AutoCal.git
   ```
2. Navigate to the project directory:
   ```sh
   cd calen
   ```
3. Install the dependencies:
   ```sh
   npm install
   ```

### Usage

To start the application, run the following command:
```sh
npm start
```
Open your browser and navigate to `http://localhost:5713` to view the application.
